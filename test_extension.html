Flow Extension Diagnostic Test

Extension Status

This page will test if the Flow extension is loaded and working properly.

Instructions

  1. Open Chrome and go to chrome://extensions
  2. Enable "Developer mode" (top right toggle)
  3. If Flow extension is loaded, check if there are any errors
  4. Click "service worker" link under Flow extension
  5. Look for console errors in the service worker DevTools
  6. Return to this page and run the tests
